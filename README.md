# Twitter US Airline Sentiment Analysis

This project performs sentiment analysis on airline-related tweets using NLP
and Machine Learning techniques.

## Features
- Text preprocessing using regex and NLTK
- TF-IDF feature extraction
- Logistic Regression classifier
- Model evaluation with confusion matrix and accuracy comparison
- Real-time sentiment prediction using Streamlit

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run preprocessing:
   python module1_module2/module2_preprocessing.py
3. Train model:
   python module3/module3_feature_extraction.py
4. Run app:
   python -m streamlit run module5/app.py

## Dataset
Twitter US Airline Sentiment Dataset (Kaggle)
